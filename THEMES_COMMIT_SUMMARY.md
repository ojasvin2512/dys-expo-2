## ✨ Bold Text Feature

### **How It Works**
Each bold word cycles through 6 theme-specific colors:

**Example in Halloween:**
```
**Spooky** **scary** **fun** **night** **treats** **magic**
```
- "Spooky" → Orange
- "scary" → Bright Orange
- "fun" → Purple
- "night" → Light Purple
- "treats" → Amber
- "magic" → Lavender

**Example in Tokyo:**
```
**Neon** **lights** **city** **night** **cyber** **future**
```
- "Neon" → Hot Pink
- "lights" → Cyan
- "city" → Yellow
- "night" → Green
- "cyber" → Orange
- "future" → Purple

**Example in Pokemon:**
```
**Fire** **Water** **Electric** **Grass** **Fighting** **Psychic**
```
- "Fire" → Red
- "Water" → Blue
- "Electric" → Yellow
- "Grass" → Green
- "Fighting" → Orange
- "Psychic" → Purple
